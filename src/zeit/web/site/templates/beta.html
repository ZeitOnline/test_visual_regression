Wir entwickeln die neue Version von ZEIT ONLINE gemeinsam mit unseren Nutzerinnen und Nutzern.

Unsere neue Site befindet sich derzeit im Beta-Test: Sie sieht an vielen Stellen noch unfertig aus, wichtige Features fehlen oder sind defekt, manchmal ist sie nicht erreichbar. Dennoch freuen wir uns, wenn Sie sie schon einmal ausprobieren wollen.

Es gibt zwei Möglichkeiten, unsere neue Seite zu testen:

1. Sie können sich für den Beta-Test bewerben. Senden Sie zu diesem Zweck bitte eine E-Mail mit Ihrem Nutzernamen oder der E-Mail-Adresse, mit der Sie in der ZEIT‐ONLINE‐Community angemeldet sind, an beta@zeit.de.

Sollten Sie noch nicht angemeldet sein, können Sie dies bei https://meine.zeit.de nachholen.

Sie erhalten eine E-Mail, sobald wir Sie für den Beta-Test freigeschaltet haben. Auf dieser Seite können Sie dann zwischen neuer und alter Version hin- und herschalten.

2. Ein Teil unserer Nutzer wird derzeit zufällig für den Beta-Test ausgewählt. Sie erhalten dann direkt auf der Homepage einen entsprechenden Hinweis.
